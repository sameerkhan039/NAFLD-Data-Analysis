OBJECTIVE:
The primary objective of this project is to analyze a medical dataset related to Non-Alcoholic Fatty Liver Disease (NAFLD). The project focuses on cleaning the data, identifying key trends, and utilizing analytical techniques to uncover potential risk factors associated with the disease.

TOOLS AND TECHNOLOGIES:
The following tools were used to complete this project:

Python: Core programming language used for analysis.

Pandas: Utilized for efficient data cleaning and manipulation.

NumPy: Used for numerical and mathematical computations.

Jupyter Notebook: Used for environment setup and documentation of the analysis process.

KEY ANALYSIS:
Performed data preprocessing to handle missing values and prepare the dataset for analysis.

Cleaned the dataset by handling missing values in key health indicators, ensuring data integrity for further analysis.

Created visualizations to simplify complex data trends and gain actionable insights.
